## Development Skills

When you need to understand or design an API, use the `telamon.rest_conventions` skill.
When you need to do anything related to architecture, use the `telamon.architecture_rules` skill.
When you need to understand the project architecture and structure, use the `telamon.explicit_architecture` skill.
When you need to write any PHP code, use the `telamon.php_rules` skill.
When you need to write PHP code in a Laravel project, use the `telamon.laravel` skill.
When you need to write PHP code in a project using the Message Bus, use the `telamon.message_bus` skill.
When you need to write documentation, use the `telamon.documentation_rules` skill.
When you need to use git, use the `telamon.git_rules` skill.
When you need to start/stop the development environment or run any command in the application container, use the `telamon.makefile` skill.
When you need to write tests or run the application test suite, use the `telamon.testing` skill.
